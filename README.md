# TaskForge 🚀

TaskForge is a lightweight task and project management app built with **React**, **TypeScript**, and **Vite**. It helps you organize tasks, manage projects, and work with multiple users in a simple interface.

---

## 📦 Features

✔ React + TypeScript  
✔ Simple user login (no backend)  
✔ Local dummy data  
✔ Project & task views  
✔ Minimal Material‑UI (MUI) interface  
✔ Fast development server with Vite

---
